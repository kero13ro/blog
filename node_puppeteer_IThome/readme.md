### 為何使用 node.js / Puppeteer，而不是 python / Selenium？

|            | Puppeteer | Selenium       |
| ---------- | --------- | -------------- |
| 支援瀏覽器 | Chromium  | 多種瀏覽器     |
| 支援語言   | Node.js   | 多種語言皆適用 |
| 安裝       | 方便快速  | 稍微麻煩       |
| 使用       | 易上手    | 想較難上手     |
